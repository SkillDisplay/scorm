SEO (Technical)

Description

The editor is aware of the relevance of duplicate content on websites and the consequences for search engine results. By using canonical links properly, an editor knows how to prevent duplicate content issues. Furthermore, the editor has knowledge about forcing robots not to index the content of a page, and/or not scan it for links to follow. The editor knows what an XML Sitemap is and that these are used to give search engines information of the websites' structure.

Goals

  • I know what canonical links are and when they should be used.
  • I know about robot instructions in metadata.
  • I know the relevance of an XML sitemap of a website.
